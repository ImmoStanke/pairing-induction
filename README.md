# Roman Numerals Kata
Convert decimal numbers into Roman Numeral notation. You may assume all numbers are positive and smaller 2^31


## Examples:

1 -> „I“

2 -> „II“

4 -> „IV“

5 -> „V“

9 -> „IX“

10 -> „X“

42 -> „XLII“

99 -> „XCIX“

2013 -> „MMXIII“
